# Features on the homepage

---

## Main features

### Class-based checkouts

Setup repeatable classes that pass the relevant date and data onto the stripe checkout.
No need to build generic stripe checkouts and products that do not have the relevant details per class. This plugin will automatically generate the correct stripe checkout with all class details (including custom data).


### Stripe checkout

Remove the hassle of handling stripe components and custom code to take credit-card data. Pass the responsibility onto stripe and stay secure in the knowledge you are not handling customer credit-card data.

### Repeating Classes, One-off Events and Regular Appointments.

Run regular repeating classes on any daily schedule, Organise one-off events throughout the year, or take complex recurring appointment schedules.
Very customisable schedules, from simple weekly repeating classes to appointments with stackable availability rules.

### Emails

As simple or complex as you want to make it. With standard admin and customer emails alongside optional reminder and post-class emails that can created using merge-tags in the standard editor, or upload HTML for full email customisation.
Overrides to all email settings on a per-class basis allows for fine-grained control of your notifications and messaging.

### Reports

At-a-glance summary reports and graphs that show upcoming class bookings as well as trends.
Determine your best customers, classes and revenue. Get class-by-class details and summaries.

### Singlular forms & Global Calendar

Place individual class forms onto your pages using customisable shortcodes or elementor widgets.
You also have a global calendar that can show multiple class types at the same time.

### Customisable

Using our theme libaray and tutorial examples you have unlimited ways to customise the forms and calendars. 
Completely restyle and design to your liking. A fully documented hook library of actions, filters and API endpoints.
Also customise using the built-in ACF library by adding your own data fields onto the forms.



## Example use-cases

### Yoga studio with regular classes

You run weekly yoga classes on monday morning 9am, tuesday evening 6:30pm and friday evenings 6pm. You take a set fee and the classes have a capacity of 12 people in your studio. This would be the text-book setup of class bookings and creating three individual classes would take no time at all. 

### Dentist with appointment slots

A busy dentist surgery that takes individual appointments for one person at multiple days/times of the week. Setup a schedule that allows you to take bookings easily with the stackable availability slots that can be repeatable or one-off.

### Dance club with one-off events

You want to run an event for your dance club. By using the one-off event type you can setup a single payment checkout that will allow you to take money for the event quickly and securely.

### PT for private gyms

You work for multiple gyms and need payment to go through those gyms systems. Simple switch on the 'external link' and point to their website.
