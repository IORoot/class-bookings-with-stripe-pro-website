## What the classes view shows

**Reports → Classes** focuses on one class at a time. Choose a class from the selector (top right) to load charts and a summary line.

![Per-class reports charts](/docs/images/reports-classes.png)

## Summary line

Shows key details for the selected class: schedule type, typical time, price, capacity, and aggregate stats for the reporting window.

## Charts

| Chart | What it shows |
|-------|----------------|
| **Students per month** | Headcount of booked students on each session date |
| **Revenue** | Money collected per session (paid bookings) |
| **Occupancy per session** | How full each session was vs capacity |
| **Cumulative revenue** | Running total earnings over time |
| **Bookings by day of week** | Which weekdays are most popular (recurring classes) |

Data is based on **paid** bookings. Currency matches your Stripe currency setting.

## Tips

- Compare occupancy to capacity to spot undersubscribed time slots.
- Use day-of-week chart when considering new weekly class times.
- Cross-check with [Bookings list](/docs/bookings-list) for individual customer detail.
