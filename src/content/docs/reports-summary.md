## What the summary view shows

**Stripe Class Pro → Reports → Summary** is the studio dashboard: how many people booked per class in a year, who is coming to upcoming sessions, and per-class guest lists for the next three sessions.

![Reports summary dashboard](/docs/images/report-summary.png)

## Bookings by class graph

- Bar or chart of **participants per class** for the selected **year**
- Use the year selector to compare seasons
- Chart supports zoom/pan on supported browsers (Chart.js)

Counts include **paid** bookings attributed to class session dates in that year.

## Booked people for upcoming classes

A quick-glance table of **upcoming classes** and how many people are booked — useful before opening individual class reports.

## Next three sessions per class

For each active class, lists **who is booked** into the next three scheduled sessions. Handy for printing door lists or checking names before class starts.

## Caching

Report aggregates are cached briefly (10 minutes) for performance. Saving a booking invalidates the cache.
