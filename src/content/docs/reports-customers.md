## What the customers view shows

**Reports → Customers** is your CRM-style list of everyone who has paid for a class: search, sort, export, and drill into their bookings.

![Reports customers table](/docs/images/report-customers.png)

## Summary line

Aggregate counts at the top — total customers, bookings, and revenue in the current dataset (updates with search/filter).

## Search

Filter the table by name or email. Results update client-side for fast browsing.

## Download CSV

Export the current customer list for spreadsheets or external tools. Columns match the visible table (name, email, booking counts, totals, last booking date).

## Table columns

Sort by clicking headers. Typical columns:

| Column | Description |
|--------|-------------|
| **Customer** | Display name |
| **Email** | Click to open mail client |
| **Bookings** | Number of paid bookings |
| **Total spent** | Sum of booking amounts |
| **Last booking** | Most recent class date |

## View bookings

Each row links to **Bookings** filtered to that customer’s email — see [Bookings list](/docs/bookings-list).
